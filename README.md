# AI TCP Game Room
[aitcpgameroom.com](http://aitcpgameroom.com)

Detailed documentation available on the Linux man page.
## To Install Man Page on Linux:
1. Copy the man page to the system man directory (requires sudo): 
`sudo cp man/ai-tcp-game-room.1 /usr/local/share/man/man1/`
2. Update the man database: 
`sudo mandb`
3. Verify by opening the manual: 
`man ai-tcp-game-room`
